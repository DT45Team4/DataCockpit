# 代码同步后各种情况对应处理
## 没有冲突时
 1. Add to Index --> 添加索引（标记新建的文件要做版本控制）
 - Commit --> 提交修改的代码到本地代码库
 - Push ---> 推送代码到远程服务器中

## 发生冲突时：（出现红色小图标）
 1. fetch --> 拉取服务器上最新代码到本机
 - merge --> 合并最新代码，到本地代码库
 - <font color=red>注意红色冲的文件；人工做代码合并处理</font>
 - Add to Index --> 添加索引（标明有修改）
 - Commit --> 提交修改后的代码到本地代码库
 - Push ---> 推送代码到远程服务器中
